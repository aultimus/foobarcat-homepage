Projects

Sextant — The Style Signal Calibration Tool

Available at sextantapp.io

Sextant is a tool for analysing and refining personal style as a signal system. Sextant provides structured feedback on coherence, friction, and alignment, alongside targeted adjustments. The underlying theory is explored in my essay The Signal and the Self. A technical write-up on building the MVP will follow.

Baconometer — An Interactive Film Network Exploration Tool

Available at baconometer.foobarcat.com

A generalisation of the Six Degrees of Kevin Bacon game, allowing any two actors to be connected via the shortest chain of shared film appearances. The Baconometer is an interactive graph traversal tool that computes the minimal co-appearance path between actors. Built to explore graph-based systems, experiment with AI-assisted development, and investigate network structure.

Janus — An Experimental Cognitive-Systems Tool

Janus is a cognitive-systems tool currently in the research phase, exploring an architectural approach to personality that focuses on cognitive structure rather than trait measurement.
